# ofertas
Clon del sitio de anuncios clasificados de Cuba, www.ofertas.cu usando Wagtail, un CMS basado en Django

# Instalación
```bash
git clone --recursive --depth 1 https://github.com/codeshard/ofertas ofertas
cd ofertas
pip install -r requeriments.txt
python manage.py runserver
```

Abrir el navegador en la dirección http://127.0.0.1:8000
